# Nike Footwear Landing Page
Одностраничный промо‑сайт, созданный по макету из Figma с использованием чистой вёрстки на HTML, CSS и JavaScript. Проект демонстрирует современный UI, акцент на визуальный контент и адаптивную структуру, ориентированную на e‑commerce сегмент.

## Описание проекта
Landing page посвящён линейке обуви Nike и включает несколько тематических секций: промо‑баннер, подборки товаров, карточки моделей, промо‑блоки и CTA‑элементы. Дизайн выполнен в тёмной палитре с акцентами красного цвета, что подчёркивает премиальность бренда.

### Функциональность
* Адаптивная вёрстка под все устройства

* Интерактивные элементы на JavaScript (переключение вариантов, hover‑эффекты и т.д.)

* Чистая структура без сторонних библиотек

* Семантическая HTML‑разметка

* Оптимизированные изображения

* Грамотная организация CSS (модули, переиспользуемые классы)

### Используемые технологии
1. HTML5 — семантическая разметка

2. CSS3 — адаптивная сетка, анимации, стилизация

3. JavaScript (Vanilla) — интерактивность и динамические элементы

### Структура проекта

```
/css        — стили проекта
/fonts      — шрифты
/img        — изображения и графика
/js         — скрипты
index.html  — основная страница
README.md   — описание проекта
```
### Цели проекта
* Практика чистой вёрстки без библиотек

* Создание современного e‑commerce лендинга

* Отработка адаптивности и UI‑детализации

* Добавление качественного проекта в портфолио
